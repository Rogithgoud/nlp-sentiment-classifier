# Sentiment Classifier on Amazon Fine Food Reviews 

This project is a simple **Naive Bayes Sentiment Classifier** built from scratch using:
- Frequency-based feature extraction
- Log priors & log likelihoods
- Amazon Fine Food Reviews dataset (10K samples)

### Features
- Preprocess reviews (tokenization, stopword removal, stemming)
- Build frequency dictionary
- Train Naive Bayes classifier
- Predict and evaluate on unseen reviews

###  Results
- Accuracy: ~78% on test data

###  Tech Stack
- Python
- NumPy
- Scikit-learn (for splitting data)
